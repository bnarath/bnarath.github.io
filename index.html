Neural Machine Translation: English → Indic Languages with Transformers & BPE (Built from Scratch)

After being fascinated by the recent advances in large language models and machine translation, I set out to build something ambitious — a Transformer-based English-to-Indic translator, implemented entirely from scratch in PyTorch. Instead of relying on high-level libraries, I built the complete sequence-to-sequence architecture from the ground up: custom attention mechanisms, positional encodings, and a Byte Pair Encoding (BPE) tokenizer trained on a parallel English-Malayalam corpus. The model was trained on GPU using PyTorch to handle the complexity of deep architectures and large vocabulary sizes. This project helped me deeply understand how transformers work under the hood, and opened the door to building more foundational NLP systems.
